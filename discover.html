Question-1: What are the disserent ways to select an element in the DOM?

JavaScript provides six methods to select an element from the document. getElementById(), getElementsByTagName(), getElementsByClassName(), getElementsByName()

Question-2: What is the difference between innerHTML, innerText and textContent?

In HTML, innerText and textContent are properties of the DOM (Document Object Model) that allow you to read and update the content of HTML elements.The innerText property focuses on the rendered text content. When you use innerText to read the content of an element, it returns the text as it appears on the screen, ignoring HTML tags.

Question-3: What is event delegation in the DOM?

Event delegation is a powerful pattern in JavaScript that leverages the concept of event bubbling to handle events more efficiently. Instead of adding an event listener to each individual element, you can add a single event listener to a parent element and manage the events for its child elements through this parent.

Question-4: What is event bubbling in the DOM?

Event bubbling is a concept in the Document Object Model (DOM) where an event triggered on an element propagates up through its parent and ancestor elements until it reaches the root element. This is the default behavior for most events in JavaScript.
